# Easton Romero — Game Music Producer

Portfolio site for Easton Romero, game music producer and composer.

## Getting started

```sh
git clone https://github.com/jkwongofficial-ctrl/rhythm-realm-demos-42f72eaf.git
cd rhythm-realm-demos-42f72eaf
npm i
npm run dev
```

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

Deployed via [Vercel](https://vercel.com), tracking the `main` branch.
